    # Pasta de Uploads

Esta pasta é usada para armazenar temporariamente os arquivos Excel importados.

## Permissões:
- A pasta deve ter permissão de escrita pelo servidor web
- Os arquivos são automaticamente removidos após o processamento

## Estrutura:
uploads/
├── temp_file_1.xlsx (processado e removido)
└── temp_file_2.xlsx (processado e removido)
