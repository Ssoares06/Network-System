# Pasta de Imagens

Esta pasta deve conter as imagens do sistema:

- logo.png - Logo da empresa (100x40px)
- favicon.ico - Ícone do site (32x32px)
- network-bg.jpg - Imagem de background (opcional)

## Estrutura sugerida:
static/img/
├── logo.png
├── favicon.ico
├── network-bg.jpg
└── icons/
    ├── switch.png
    ├── server.png
    └── network.png
